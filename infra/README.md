# Infrastructure

Este directorio contiene archivos de configuración de infraestructura para el proyecto Fantasy Life.

## Contenido

- Configuración de Nginx (futuro)
- Scripts de deployment (futuro)
- Configuraciones de CI/CD (futuro)

## Docker

La configuración principal de Docker está en el archivo `docker-compose.yml` en la raíz del proyecto.

Para iniciar el proyecto:

```bash
docker compose up --build
```
